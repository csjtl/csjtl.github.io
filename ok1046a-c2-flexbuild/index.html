ејҖеҸ‘жқҝеҒңз•ҷеңЁ uboot е‘Ҫд»ӨиЎҢ
й…ҚзҪ®ејҖеҸ‘жқҝ IP setenv ipaddr 10.1.1.111
й…ҚзҪ® Server IP setenv serverip 10.1.1.100
Ping жөӢиҜ• ping 10.1.1.100
дёӢиҪҪж–Үд»¶еҲ°еҶ…еӯҳ tftp 0x90000000 firmware_ls1046ardb_uboot_qspiboot_1133_5a59.img
жЈҖжөӢQSPI FLASH sf probe 0:0
ж“ҰйҷӨжҢҮе®ҡдҪҚзҪ®,жҢҮе®ҡй•ҝеәҰзҡ„flashеҶ…е®№ sf erase 0 0x1000000
еҶҷеҶ…еӯҳж•°жҚ®еҲ°flash sf write 0x90000000 0 0x1000000
йҮҚеҗҜ reset
